# CLI操作 学習計画

CLI（コマンドラインインターフェース）の基本操作から応用的な検索・フィルタリング、Git操作までを体系的に学習します。

## 🎯 学習ゴール
1. 基本的なファイル・ディレクトリ操作をスムーズに行えるようになる
2. パイプやフィルタ（grep等）を活用した情報の検索・抽出ができるようになる
3. コマンドラインを用いた効率的なワークフローを確立する

---

## 🛠 使用技術スタック
- PowerShell (Windows環境)
- Git

---

## 📅 カリキュラム構成

### 【Phase 1】 基本操作
- ディレクトリ移動 (cd) と一覧表示 (ls/dir)
- ファイル・ディレクトリの作成・削除 (mkdir, ni, rm)
- ファイルのコピー・移動 (cp, mv)

### 【Phase 2】 検索とフィルタリング
- 文字列検索 (Select-String / grep)
- ファイル検索 (Get-ChildItem -Recurse / find)
- パイプライン (|) を使ったコマンドの組み合わせ

### 【Phase 3】 Git操作
- リポジトリの初期化とステータス確認
- コミット、ログの確認
- ブランチ操作
